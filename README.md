## ✨ Overview

ChatFlow is a full-stack real-time messaging platform inspired by modern chat applications. The goal is to build a scalable messaging system while learning backend architecture, authentication, databases, and real-time communication.

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- TypeScript

### Backend
- Express.js
- Socket.IO
- JWT Authentication
- Bcrypt

### Database
- PostgreSQL
- Prisma ORM

### Deployment
- Vercel
- Railway / Render
- Neon PostgreSQL
